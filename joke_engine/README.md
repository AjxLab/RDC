# ダジャレ判定エンジン


## Requiremenst
- Python 3
- Django 3.0.3


## Usage
### Build
#### Fetch All Jokes Data
```sh
$ python build.py fetch
```
#### Train
```sh
$ python build.py train
```
### Django
#### Run Server
```sh
$ python manage.py runserver
```
#### Run Test
```sh
$ python manage.py test
```


## Installation
### Clone this repo
```sh
$ git clone https://github.com/AjxLab/RDC
$ cd RDC/joke-engine
```
### Install python libs
```sh
$ pip install -r lib/requirements.txt
```
